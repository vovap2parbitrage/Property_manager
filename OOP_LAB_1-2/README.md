#Property management system
This project is a C++ Object-Oriented Programing application that simulate real estate management system. It demonstrates **Separate Compilation** by spliting classes into headers (.h) and source (.cpp) files

##How to build and run
This project is configured for **Visual Studio Code** using the MinGW (g++) compiler.

### Prerequisites
* Visual Studio Code
* MinGW/g++ compiler installed and added to system PATH.

#### Build Instructions
1.  Open the project folder in VS Code.
2.  Open any `.cpp` file (e.g., `property_management_lab_1.cpp`).
3.  Press **`Ctrl` + `Shift` + `B`** to build the project.
4.  Enter in terminal ".\<filename>.exe"
